Cryptography

Completed Cryptography through Code Labs Academy (CLA).

üîë Key Concepts

  • Definition: Study of mathematical techniques for securing digital information against adversarial attacks.
  • Daily Use: Employed for privacy and encrypting messages with keys for confidentiality. Example: WhatsApp encrypts conversations to prevent hacking or interception.
  • Importance of Cryptography: Integral to modern life; without it, bank transactions, internet traffic, and cell phones fail, exposing sensitive information.
  • Cryptographic Algorithms Types: Includes Symmetric-key, Asymmetric-key, Stream Ciphers, and Block Ciphers.
  • Symmetric-key Algorithms: Uses a single key for both encryption and decryption. Examples: AES, DES, 3DES.
  • Asymmetric-key Algorithms: Uses separate keys for encryption and decryption. Examples: RSA, Elliptic Curve Cryptography (ECC).
  • Stream Ciphers: Encrypt one bit or byte at a time with dynamic key changes. Examples: A5/1, RC4.
  • Block Ciphers: Encrypt fixed-size blocks of data. Example: Feistel Cipher.
  • Classical Cryptography: Focuses on designing and using codes (ciphers) for secure communication. Examples: Caesar Cipher, Rail Fence Cipher.
  • Modern Cryptography: Involves mathematical methods to secure communication. Objectives include confidentiality, integrity, authenticity, and non-repudiation.
  • Hashing: Converts data into a fixed-size hash value. Examples: MD5, SHA-1, SHA-2, SHA-3.
